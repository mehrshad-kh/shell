# NAME
**genman.sh** - generate man page from Markdown via pandoc

# SYNOPSIS
**genman.sh** _file_ _output_directory_

# DESCRIPTION
**genman.sh** takes a Markdown source as _file_ and creates the corresponding 
man page in _output_directory_.
