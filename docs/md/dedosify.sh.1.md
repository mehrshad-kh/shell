# NAME
**dedosify.sh** - convert Windows (DOS) text file to UNIX

# SYNOPSIS
**dedosify.sh** _file_

# DESCRIPTION
**dedosify.sh** replaces _file_ with the equivalent with CRLF line
terminators converted to LF and tabs expanded to four spaces.

# HISTORY
Started in May 2024.
