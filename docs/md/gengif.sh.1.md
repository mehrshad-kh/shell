# NAME
**gengif.sh** - generate GIF from video

# SYNOPSIS
**gengif.sh** -t _duration_ _input_ _output_

# DESCRIPTION
**gengif.sh** takes in a video as _input_ and creates an _output_ 
of _duration_ seconds.

# REQUIREMENTS
+ ffmpeg
